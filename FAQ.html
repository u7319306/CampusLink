Frequently Asked Questions

Q1. What is this directory for?

It helps ANU students quickly find official campus services in one place (e.g. wellbeing, academics, admin, careers, clubs).

Q2. What types of services are included in this directory?

We focus on official ANU student services (e.g. health, wellbeing, academic support, admin, careers, clubs). External or private services are not listed.

Q3. How do I find a service?

Use search or filter by category. Each service page shows a short description, contact info, location and how to access it.

Q4. Can I change the language of the website?

Yes, use the language toggle at the top of the page. More languages will be added over time.

Q5. Is the information always up to date?

We try to keep all pages accurate, but some services change without notice. If something looks outdated, you can report it.

Q7. Does this website replace Wattle or ANU systems?

No, this is just a directory. It links you to the official service pages so you can take action there.

Q8. Can I book appointments through this site?

Not directly. We give you the booking link or contact method for each service.

Q9. Why are only a few languages available?

To launch faster, we started with core languages. More will be added once the main features are stable.

Q10. Is this an official ANU website?

This is a student-built project created for ANU students. Some information links to official ANU services, but the directory itself is developed independently as a student initiative.
